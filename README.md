# Critter Chronologer

## Overview
Critter Chronologer is a backend scheduling and tracking system developed as part of the
Udacity Data Stores and Persistence course under Accenture training.
The application manages pets, employees, schedules, and availability for a pet service business.

## Problem Statement
The goal of this project was to design and implement a RESTful backend application
that supports persistent data storage, complex entity relationships, and API-based access.

## Key Features
- Employee and pet management
- Scheduling and availability tracking
- REST APIs using Spring Boot
- Database persistence using JPA and Hibernate

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- REST APIs
- H2 / MySQL

## How to Run
1. Clone the repository
2. Open the project in an IDE (IntelliJ / Eclipse)
3. Build the project using Maven
4. Run the Spring Boot application
5. Access APIs using Postman or browser
